## General

- Post titles should be in title case.

- There is a preference for U.S. English with noted exceptions.

  - Any word that has an "ou" in UK English (e.g. colour or neighbour)
  - Any "double-l" word (e.g. travelled or cancelled)

## Abbreviations

- Unless otherwise noted, no abbreviation should contain periods.

  The "United States of America" should only be abbreviated as "U.S." or "U.S.A.".

## Punctuation and syntax

- The Oxford comma is required in lists of three or more items.
- All code should be rendered in code blocks.

  Code spanning less than one line may be rendered in an inline block.
  All others must be rendered in a block with the [appropriate language](https://shiki.style/languages) provided.
