## Daily Drivers

- **MacBook Pro** (2018, 15-inch)

  I still daily drive a 2018 MacBook Pro, and aside from a few storage issues (I need to free up space), it still runs great.

  *Intel Core i7, 16 GB RAM, 500 GB SSD — space grey*

- **Pixel 7**

  The Pixel version of Android is so clean.
  The only problem I have is that the [September 2025 Pixel Drop](https://www.androidauthority.com/google-pixel-drop-september-2025-3593796) uses way more battery than previously: I used to be able to charge my phone overnight and use it basically all day, but now I have to charge it after a few hours.

  *256 GB — Lemongrass*

- **OnePlus Watch 2**

  It's a great watch.

- **YubiKey 5**

  I have multifactor authentication enabled on all the services I use.
  I also help maintain the [2FA Directory](https://2fa.directory) and the [Passkeys Directory](https://passkeys.2fa.directory).

## Development

- [**Neovim**](https://neovim.io)

  I got fed up with Visual Studio Code and went all in on Neovim.
  If you're interested, my (messy) configuration is in [my dotfiles repository](https://github.com/hkamran80/dotfiles/tree/main/.config/nvim).

- [**WezTerm**](https://wezterm.org)

  I tried the native macOS Terminal, iTerm 2, and Hyper before settling on WezTerm.

- [**Homebrew**](https://brew.sh)

  Homebrew is one of the first pieces of software I install on any macOS or Linux machine.

- **`zsh`**

  It's better than `bash` in my opinion.
  My configuration is in [my dotfiles repository](https://github.com/hkamran80/dotfiles/blob/main/.zshrc).

## Browsing

My browser of choice is [**Firefox**](https://firefox.com).
Even though they are [going all in on "AI"](https://www.windowscentral.com/software-apps/mozilla-says-firefox-will-evolve-into-an-ai-browser-and-nobody-is-happy-about-it-ive-never-seen-a-company-so-astoundingly-out-of-touch) (ugh), I still far prefer it to any Chromium-based browser or Safari.

### Extensions

- [Bitwarden](https://bitwarden.com/download/#downloads-web-browser)
- [uBlock Origin](https://addons.mozilla.org/firefox/addon/ublock-origin)
- [Dark Reader](https://darkreader.org) (dark mode everything)
- [DuckDuckGo Privacy Essentials](https://addons.mozilla.org/firefox/addon/duckduckgo-for-firefox)
- [RSSHub Radar](https://github.com/DIYgod/RSSHub-Radar) (helps find RSS/Atom feeds)
- [Violentmonkey](https://violentmonkey.github.io) (userscript manager)
- [SponsorBlock](https://sponsor.ajay.app)
- [Return YouTube Dislike](https://www.returnyoutubedislike.com)
- [Firefox Multi-Account Containers](https://addons.mozilla.org/firefox/addon/multi-account-containers) (amazing for logging into the same service with multiple accounts)
- [Nebula New Tab](/projects/nebula-new-tab) (my new tab extension)
- [TOS;DR](https://tosdr.org) (simplified version of most Terms of Service)
